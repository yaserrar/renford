# Comment Renford trouve les bons profils

Ce document explique, en version simple, comment la plateforme propose les meilleurs Renfords pour une mission.

## Le but

Quand un etablissement publie une mission, Renford cherche les profils les plus adaptes pour:

1. gagner du temps,
2. proposer des candidats vraiment compatibles,
3. eviter les propositions inutiles.

## Quand la recherche se lance

La recherche se fait automatiquement:

1. au moment de la publication d'une mission,
2. ensuite a intervalles reguliers,
3. et aussi quand un candidat refuse, pour proposer d'autres options.

## Comment un profil est retenu

Avant tout, la plateforme verifie si le profil peut vraiment faire la mission.

Elle regarde notamment:

1. le type de mission,
2. le niveau d'experience,
3. les disponibilites sur les jours et horaires,
4. la distance et la mobilite,
5. la compatibilite de tarif,
6. les conditions obligatoires demandees par la mission.

Si un point important ne correspond pas, le profil n'est pas retenu.

## Comment les profils sont classes

Les profils compatibles recoivent ensuite un score.

Ce score tient compte de plusieurs elements, par exemple:

1. la proximite geographique,
2. l'experience,
3. le parcours professionnel,
4. les diplomes,
5. l'adequation au budget,
6. la qualite generale du profil.

Ensuite, la plateforme trie les profils du plus adapte au moins adapte.

## Ce qui se passe apres le classement

Une fois le classement fait:

1. les meilleurs profils sont proposes,
2. les anciennes propositions moins pertinentes peuvent etre retirees,
3. les candidatures deja bien avancees ne sont pas cassees.

## Notifications automatiques

La plateforme envoie des notifications dans les moments importants:

1. quand une nouvelle mission correspond a un Renford,
2. quand un Renford accepte une mission,
3. quand un etablissement retient un candidat.

## En cas de refus

Si un Renford refuse une mission:

1. la plateforme relance une recherche,
2. elle propose d'autres profils compatibles.

Si un etablissement refuse un candidat:

1. la liste est mise a jour,
2. la recherche continue jusqu'a trouver un bon profil.

## Resume rapide

Le systeme Renford:

1. filtre les profils pour ne garder que les compatibles,
2. classe les meilleurs candidats,
3. propose jusqu'a 50 profils,
4. met a jour les propositions automatiquement,
5. protege les candidatures deja engagees.
